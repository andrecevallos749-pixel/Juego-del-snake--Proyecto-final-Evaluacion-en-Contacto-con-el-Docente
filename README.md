# Objetivo
Desarrollar un videojuego tipo Snake utilizando Python y la biblioteca Pygame, aplicando los conocimientos adquiridos durante la asignatura, tales como estructuras condicionales, estructuras repetitivas, listas, funciones, operadores lógicos, validación de datos y control de versiones mediante Git y GitHub.

# Descripción del proyecto
 Este sistema, proyecto final de la asignatura de Lógica de Programación, busca demostrar la aplicación práctica de los contenidos estudiados durante el curso mediante el desarrollo de un software funcional e interactivo. El jugador controla una serpiente que debe alimentarse para aumentar su puntaje mientras evita colisionar con las paredes o con su propio cuerpo que va creciendo. Además del funcionamiento clásico del juego, se incorporaron diferentes tipos de alimentos con efectos especiales (velocidad, escudo, trampa),  comida normal, un sistema de selección de dificultad (Fácil, medio, y difícil), pantallas de navegación y control de estados del juego.

# Funcionalidades
- Menú principal.
- Selección de dificultad (Fácil, Medio y Difícil).
- Movimiento de la serpiente mediante el teclado.
- Sistema de puntaje.
- Crecimiento de la serpiente.
- Generación aleatoria de alimentos.
- Cuatro tipos de comida con efectos diferentes:
    - Comida normal (+1 punto).
    - Comida dorada (+5 puntos).
    - Comida trampa (-1 punto y reduce el tamaño de la serpiente)
    - Comida de velocidad (aumento temporal de velocidad).
    - Comida de escudo (invulnerabilidad temporal).
- Incremento automático de velocidad cada cinco puntos.
- Detección de colisiones con paredes y cuerpo.
- Pantalla de Game Over.
- Opción para reintentar la partida.
- Cambio de dificultad.
- Cierre del juego mediante pantalla de despedida.
- Control de versiones utilizando Git y GitHub.
